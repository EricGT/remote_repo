MGU

Explanation
a = a Succeeds {}
a = b Fails {} a and b do not match
X = X Succeeds {}
a = X Succeeds { X ↦ a } X is unified with the constant a
X = Y Succeeds { X ↦ Y } X and Y are aliased
f(a,X) = f(a,b) Succeeds { X ↦ b } function and constant symbols match, X is unified with the constant b
f(a) = g(a) Fails {} f and g do not match
f(X) = f(Y) Fails { X ↦ Y } X and Y are aliased
f(X) = g(Y) Fails {} f and g do not match
f(X) = f(Y,Z) Fails {} Fails. The f function symbols have different arity
f(g(X)) = f(Y) Succeeds { Y ↦ g(X) } Unifies y with the g(x)
f(g(X),X) = f(Y,a) Succeeds { X ↦ a, Y ↦ g(a) } Unifies X with constant a, and Y with the term g(a)
X = f(X) Succeeds
Should be Fails. 		{ X ↦ f(f(f(f(...)))) } Returns ⊥ in first-order logic and many modern Prolog dialects (enforced by the occurs check).
Succeeds in traditional Prolog and in Prolog II, unifying X with infinite term X=f(f(f(f(...)))).
X = Y, Y = a Succeeds { X ↦ a, Y ↦ a } Both X and Y are unified with the constant a
a = Y, X = Y Succeeds { X ↦ a, Y ↦ a } Both X and Y are unified with the constant a
X = a, b = X Fails {} Fails. a and b do not match, so x can't be unified with both

Parent and gender facts combined

Predicate: mother 
mother(Mother,Child) :-     % Mother is the mother of Child if
                            parent(Mother,Child),     % Mother is the parent of Child and
                            female(Mother).           % Mother is female

Predicate: grandparent 
grandparent(Grandparent,Grandchild) :-  % Grandparent is a grandparent of Grandchild if
                            parent(Grandparent,Parent),           % Grandparent is a parent of Parent and
                            parent(Parent,Grandchild).            % Parent is a parent of Grandchild

Predicate: sister_01 
sister_01(Sister,Sibling) :-   % Sister is a sister of Sibling if
                            parent(Parent,Sister),      % Parent is a parent of Sister and
                            parent(Parent,Sibling),     % Parent is a parent of Sibling and
                            female(Sister).             % Sister is female.

Predicate: sister 
sister(Sister,Sibling) :-     % Sister is a sister of Sibling if
                            parent(Parent,Sister),
                            parent(Parent,Sibling),     % Sister and Sibling have the same Parent and
                            female(Sister),             % Sister is female and
  Sister \= Sibling.          % Sister and Sibling are different
Note: Sister \= Sibling needs to come after the variables Sister and Sibling are bound
otherwise the comparison (needing unification) will fail.

Predicate: aunt 
aunt(Person,Aunt) :-       % A Person is an Aunt if
                            parent(Parent,Person),   % Parent is a parent of Person and
                            sister(Aunt,Parent).     % Aunt is a sister of Parent

Predicate: aunt
sister unfolded 
aunt(Person,Aunt) :-            % A Person has an Aunt if
                            parent(Parent_02,Person),     % Parent_02 is a parent of Person and
                            parent(Parent_01,Aunt),
                            parent(Parent_01,Parent_02),  % Aunt and Parent_02 have the same Parent_01 and
                            female(Aunt),                 % Aunt is female and
  Aunt \= Parent_02.            % Aunt and Parent_02 are different

Predicate: ancestor 
ancestor(X,Z) :-      % X is an ancestor of Z if
                                        parent(X,Z).        % X is a parent of Z

ancestor(X,Z) :-      % X is an ancestor of Z if
                                        parent(X,Y),        % X is a parent of Y and
                                        ancestor(Y,Z).      % Y is an ancestor of Z

Predicate: ancestor
ancestor unfolded 
ancestor(Y,Z) :-  % Y is an ancestor of Z if
                                        parent(Y,Z).    % Y is a parent of Z

ancestor(X,Z) :-    % X is an ancestor of Z if
                                        parent(X,Y),      % X is a parent of Y and
                                        parent(Y,Z).      % Y is an ancestor of Z

ancestor(W,Z) :-      % W is an ancestor of Z if
                                        parent(W,X),        % W is a parent of X and
                                        parent(X,Y),        % X is a parent of Y and
                                        parent(Y,Z).        % Y is a parent of Z

ancestor(V,Z) :-      % V is an ancestor of Z if
                                        parent(V,W),        % V is a parent of W and
                                        parent(W,X),        % W is a parent of X and
                                        parent(X,Y),        % X is a parent of Y and
                                        parent(Y,Z).        % Y is a parent of Z
